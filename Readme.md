## Description
This project is a **learning-based Machine Learning project** created to understand the working of the **Naive Bayes algorithm**.

The system classifies SMS messages into **Spam** and **Not Spam (Ham)** using **Natural Language Processing (NLP)** techniques and **Multinomial Naive Bayes** classifier.

The main goal of this project is to learn how Naive Bayes works in text classification problems.

---

## Learning Objectives
- Understand Naive Bayes algorithm  
- Learn text preprocessing using NLP  
- Convert text into numerical features using TF-IDF  
- Train and test a classification model  
- Deploy a basic ML model using Streamlit  

---

## Technologies Used
- Python  
- NLTK  
- Scikit-learn  
- Streamlit
